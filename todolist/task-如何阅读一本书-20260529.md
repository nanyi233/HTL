# 如何阅读一本书

- 🎯 **目标**：从一本书中真正吸收作者的核心观点和论证，并能用自己的话复述、应用，而不是"翻完了"。
- ⏱️ **预计总耗时**：~315min（约 5 小时 15 分，可分 5~7 天进行）
- 📦 **交付物**：一份个人读书笔记（含一句话主旨、章节结构图、重点摘录、3 条行动启发）

## 1. 选书与准备
- [ ] 1.1 在书架/微信读书/Kindle 里挑出此次要读的那一本，写下书名+作者 ⏱️ ~3min
- [ ] 1.2 用一句话写下"我为什么读这本书"（解决什么问题/学什么） ⏱️ ~5min
- [ ] 1.3 准备好笔记载体（新建一个 Notion/Obsidian 页面或一本纸质本，命名为书名） ⏱️ ~3min
- [ ] 1.4 设定本次阅读的截止时间（比如 7 天内读完） ⏱️ ~2min

## 2. 检视阅读（先看骨架，10 分钟看完一本书）
- [ ] 2.1 读封面、腰封、作者简介，记下作者背景关键词 ⏱️ ~5min
- [ ] 2.2 读目录，把章节标题抄到笔记里，标出最感兴趣的 3 章 ⏱️ ~10min
- [ ] 2.3 读序言/前言/译者序，记录作者自述的"本书要解决什么问题" ⏱️ ~10min
- [ ] 2.4 翻最后一章/结语，瞄一眼作者最终结论 ⏱️ ~8min
- [ ] 2.5 随机翻 5~10 页正文，感受写作密度，决定阅读节奏（粗读还是精读） ⏱️ ~7min
- [ ] 2.6 在笔记顶部写下 3 个你想从这本书得到答案的问题 ⏱️ ~5min

## 3. 分析阅读（带着问题精读）
- 3.1 拆分章节，单章逐个攻破
  - [ ] 3.1.1 读完一章后，合上书，用 3 句话复述本章主旨 ⏱️ ~10min
  - [ ] 3.1.2 把本章关键概念/术语写到笔记，每个用自己的话定义 ⏱️ ~10min
  - [ ] 3.1.3 摘录 1~3 句最打动你的原文，注明页码 ⏱️ ~7min
  - [ ] 3.1.4 在页边空白写一个"对我有什么用"的小注 ⏱️ ~5min
- [ ] 3.2 每读完 2~3 章，用一张图（思维导图/手绘）画出当前的全书结构 ⏱️ ~15min
- [ ] 3.3 读到反对/疑惑的地方，立刻在笔记里写"我不同意，因为…" ⏱️ ~10min
- [ ] 3.4 全书读完后，用一句话写出作者的核心主张（≤30字） ⏱️ ~10min
- [ ] 3.5 列出作者用来支撑核心主张的 3~5 个主要论据 ⏱️ ~15min

## 4. 主题串联（把这本书放进知识网）
- [ ] 4.1 写下你之前读过/听过的、与本书观点相似的 1~2 个来源 ⏱️ ~10min
- [ ] 4.2 写下与本书观点相反的 1 个来源或反例 ⏱️ ~10min
- [ ] 4.3 用一段话（5~8 句）回答开篇写下的那 3 个问题 ⏱️ ~15min

## 5. 输出与内化
- [ ] 5.1 写一段 200~300 字的"一句话推荐+亮点摘要"（朋友能看懂的程度） ⏱️ ~15min
- [ ] 5.2 提炼出 3 条可立即应用到生活/工作的行动建议 ⏱️ ~12min
- [ ] 5.3 选其中 1 条最容易做的行动，今天或明天就执行一次 ⏱️ ~15min
- [ ] 5.4 在日历上设定一个"30 天后回顾"的提醒 ⏱️ ~3min
- [ ] 5.5 给本书打分（1~10）并写下"是否推荐 / 推荐给谁" ⏱️ ~5min

---

## 🔗 依赖关系
- 必须先做 1.x 再开始 2.x（没准备好笔记载体精读会偷懒）
- 2.x 完成后再做 3.x（先看骨架再精读，效率高一倍）
- 3.x 全部完成后再做 4.x（必须读完整本才能串联）
- 5.x 在 4.x 之后做，但 5.3「立刻执行」可以并行启动

▶️ **建议从这里开始**：1.1 打开你想读的那本书，把书名+作者写到笔记第一行。
